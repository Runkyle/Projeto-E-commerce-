# SyntaxWear - E-commerce de Sneakers 👟

![Logo](./img/logo/logo.svg)

Bem-vindo ao **SyntaxWear**, um projeto de e-commerce moderno e elegante especializado em tênis e sneakers. Este projeto foi desenvolvido com foco em design responsivo, usabilidade e uma estética urbana contemporânea.

## 🚀 Sobre o Projeto

O SyntaxWear é uma plataforma de vitrine para calçados que combina tecnologia e estilo. O site foi estruturado para proporcionar uma experiência de navegação fluida, permitindo que os usuários explorem diferentes categorias de produtos e encontrem os modelos mais recentes, como a linha exclusiva **Krypton One**.

## ✨ Funcionalidades

- **Design Responsivo:** Adaptável a diferentes tamanhos de tela (desktop, tablet e mobile).
- **Hero Section Dinâmica:** Destaque para lançamentos com chamadas para ação (CTA).
- **Categorias Intuitivas:** Navegação rápida por estilos (Casual, Esporte, Moderno, Futurista).
- **Grade de Produtos:** Visualização organizada dos modelos em destaque.
- **Newsletter:** Área dedicada para captura de e-mails e fidelização.
- **Redes Sociais:** Integração visual com links para Instagram, WhatsApp, Facebook e TikTok.
- **Menu Hamburguer:** Navegação otimizada para dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3:** Estilização avançada utilizando:
    - **Variáveis CSS (Custom Properties):** Para fácil manutenção de cores e tipografia.
    - **Flexbox & CSS Grid:** Para layouts complexos e responsivos.
    - **Animações e Transições:** Para uma experiência de usuário mais interativa.
- **Google Fonts:** Tipografia selecionada (Ubuntu, Roboto, Intel One Mono) para legibilidade e estilo.

## 📁 Estrutura do Projeto

```text
├── index.html          # Página principal do site
├── css/                # Diretório de estilos (CSS Modular)
│   ├── base.css        # Estilos base e botões
│   ├── variables.css   # Definição de cores e fontes
│   ├── layout.css      # Estrutura geral do layout
│   ├── header.css      # Estilização do cabeçalho e navegação
│   ├── hero.css        # Estilos da seção principal (Hero)
│   ├── products.css    # Estilos das categorias
│   ├── products-grid.css # Grade de exibição de produtos
│   └── footer.css      # Estilização do rodapé
├── img/                # Imagens de banners, produtos e logotipos
└── icons/              # Ícones SVG utilizados na interface
```

## 💻 Como Visualizar

Para visualizar o projeto localmente:

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador de preferência.
3. Ou utilize a extensão **Live Server** no VS Code para uma visualização em tempo real.
